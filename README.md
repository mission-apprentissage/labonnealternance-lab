```shell

$ pip install -r requirements.txt
```


### Exit virtual environment
```shell
$ deactivate
```

## 2. Create docker image
### Build image
```shell
$ docker buildx build --platform linux/amd64 -t lba-classifier .
```

### Run image
```shell
docker run --rm -it -p 8000:8000 --name classifier lba-classifier
```
